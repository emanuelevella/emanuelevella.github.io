## Prerequisite

On the root of the project create a .env files that will store the api key like this:

```sh

REACT_APP_APIKEY=yourapikey

```

## Install

```sh
(yarn install)
```

## Usage

```sh
(yarn start)
```

## Run tests

```sh
(yarn run test)
```
